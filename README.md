# 📄 Views – SQL View Creation and Usage

## 📌 Objective:
To learn how to create, query, and use SQL Views for data abstraction, reusability, and security in MySQL using the `emp`, `dept`, and `salgrade` tables.

---

## 🛠 Tools Used:
- MySQL Workbench  
- DB Browser for SQLite *(if applicable)*

---

## 🗃️ Database Setup:
Created a database named `company1` with the following tables:

- `emp` – stores employee details
- `dept` – stores department info
- `salgrade` – holds salary grade ranges

---

## ✅ Views Created:

### 1. empdata  
Displays employee names and their job roles.

### 2. dept_avg_salary  
Shows average salary department-wise.

### 3. old_employees  
Lists employees who joined before 1982.

### 4. no_comm_employees  
Shows employees with no or zero commission.

---

## ✅ Outcome:

- Understood how views simplify complex queries
- Learned to secure and abstract data using views
- Practiced reusable logic through named views
